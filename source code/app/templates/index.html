Global Terrorism

Terrorism is an ever-present threat all round the world. How can we fight against terrorism? One of a few things we can do is participation and comprehension. This website provides insights on terrorism attacks, such as the most dangerous weapon and the most common victims, via the power of data analysis and various visualizations.
